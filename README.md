# Music Bot

## Features

* Search from YouTube (search/link/playlist)
* Save current playlist via database
* Play music from my computer (only for whitelisted users)
* All  controls like skip, volume, etc.
* DM music!

## Commands

* `/play <search/url>`
* `/skip`
* `/seek <time>`
* `/volume <0-100>`
* `/pause`
* `/clear`
* `/save <playlist>`
* `/load <playlist>`
* `/list playlists`
